## Installation steps

Create project:
```
$ ng new shopping-list
```

Install [Angular Material](https://material.angular.io)
```
$ ng add @angular/material
```
- Yes to all

Install [Angular Flex-Layout](https://github.com/angular/flex-layout)
```
$ npm i -s @angular/flex-layout @angular/cdk
```
